## WalkSAT



Note: all commands are executed under the `WalkSAT` directory, so before anything, run:

```bash
cd WalkSAT/
```


### Running all tests

```bash
python src/main.py
```


### Running single test

```bash
python src/main.py data/testsuit/file.cnf
```


### Running a different test suite

```bash
python src/main.py data/uf250-1065/
```

### Specifying a seed

Note: Every commands listed previously support this

```bash
python src/main.py --seed 1234
```
